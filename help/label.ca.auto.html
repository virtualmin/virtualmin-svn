Vols permetre els repositoris de subversió?
Quan aquesta opció està habilitada, el propietari d'aquest servidor virtual serà capaç de crear un o més dipòsits de Subversion associats al domini per emmagatzemar codi font controlat per a versions i altres fitxers. Els dipòsits disponibles per a cada usuari es poden configurar a la pàgina de correu / usuaris d’FTP.
