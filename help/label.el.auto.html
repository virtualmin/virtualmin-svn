Να επιτρέπονται τα αποθετήρια Subversion;
Όταν είναι ενεργοποιημένη αυτή η επιλογή, ο κάτοχος αυτού του εικονικού διακομιστή θα είναι σε θέση να δημιουργήσει ένα ή περισσότερα αποθετήρια Subversion που σχετίζονται με τον τομέα για την αποθήκευση πηγαίου κώδικα ελεγχόμενης έκδοσης και άλλων αρχείων. Τα αποθετήρια που είναι διαθέσιμα σε κάθε χρήστη μπορούν να διαμορφωθούν στη σελίδα χρηστών αλληλογραφίας / FTP.
